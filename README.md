# iPosta — Bug Reports & Feature Requests

This repository is the public issue tracker for **iPosta**, a native macOS email client for Gmail and Outlook.

[Download iPosta](https://iposta.app) | [Privacy Policy](https://iposta.app/privacy)

---

## Reporting a Bug

Before opening an issue, please check [existing issues](../../issues) to avoid duplicates.

When reporting a bug, include:

- **macOS version** (e.g., macOS 26.0)
- **iPosta version** (found in iPosta → About iPosta)
- **Email provider** (Gmail, Outlook, or both)
- **Steps to reproduce** the issue
- **Expected behavior** vs. **actual behavior**
- **Screenshots** if applicable

You can use the **Bug Report** template when creating a new issue.

## Requesting a Feature

Have an idea to make iPosta better? Open a feature request with:

- A clear description of the feature
- Why it would be useful
- Any examples from other apps (optional)

## Collecting Diagnostic Logs

If you're experiencing sync or connection issues, you can export logs from within the app:

1. Open iPosta → Settings → Diagnostics
2. Select a time period (5 / 30 / 60 min)
3. Click **Export**
4. Attach the exported file to your issue
